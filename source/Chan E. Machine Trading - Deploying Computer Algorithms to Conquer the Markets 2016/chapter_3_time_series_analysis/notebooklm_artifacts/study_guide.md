# 시계열 분석 학습 가이드

본 학습 가이드는 경제계량학과 신호 처리 분야에서 널리 활용되며, 최근 퀀트 투자 커뮤니티에서도 중요하게 다뤄지는 시계열 분석 기법을 체계적으로 학습하기 위해 제작되었습니다. 이 가이드는 AR, ARMA, ARIMA, VAR, VEC, 그리고 상태 공간 모델(State Space Model)을 포함한 선형 모델링 기법의 핵심 원리와 실전 적용 사례를 다룹니다.

---

## 1. 핵심 개념 요약

### 시계열 분석의 목적과 적용
*   **목적:** 과거의 데이터를 바탕으로 시계열의 다음 신호를 예측하는 것입니다.
*   **주요 적용 시장:** 펀더멘털 정보나 직관이 단기 예측에 부족한 외환(FX) 시장이나 비트코인 시장에서 특히 유용합니다. 주식 시장에서도 기술적 분석의 보완 도구로 활용됩니다.

### 주요 모델 유형

| 모델명 | 설명 | 핵심 특징 |
| :--- | :--- | :--- |
| **AR(p)** | 자기회귀 모델. 현재 가격을 과거 $p$개의 가격 선형 결합으로 설명함. | $| \phi | < 1$일 때 약정상성 및 평균 회귀 특성을 가짐. |
| **ARMA(p, q)** | 자기회귀 이동평균 모델. AR 모델에 $q$개의 과거 오차(innovation) 항을 추가함. | 모델의 구조적 단순함을 보완하여 필요한 시차(lag) 수를 줄임. |
| **ARIMA(p, d, q)** | 자기회귀 누적 이동평균 모델. 데이터를 $d$번 차분하여 ARMA를 적용함. | 주로 로그 수익률을 모델링할 때 사용됨 ($d=1$). |
| **VAR(p)** | 벡터 자기회귀 모델. 여러 시계열 변수 간의 상호작용을 행렬로 모델링함. | 상관관계가 있는 자산 집단(예: 동일 업종 주식들) 분석에 적합함. |
| **VEC(q)** | 벡터 오차 수정 모델. VAR 모델을 차분 변수와 수준 변수의 결합으로 변형함. | 변수 간의 장기적인 관계(공적분)와 단기적 수정을 동시에 분석함. |
| **상태 공간 모델 (SSM)** | 관찰되지 않는 '은닉 상태(Hidden State)'가 관찰 변수를 결정한다고 가정함. | 칼만 필터(Kalman Filter)가 대표적인 예시임. |

### 모델 선택 및 평가 지표
*   **BIC (Bayesian Information Criterion):** 모델의 복잡도(파라미터 수)를 페널티로 부여하여 최적의 시차($p, q$)를 결정하는 지표입니다. BIC 값이 낮을수록 모델이 적합하다고 판단합니다.
*   **MLE (Maximum Likelihood Estimation):** 주어진 데이터에 대해 모델의 파라미터가 나타날 확률을 최대화하는 방식으로 계수를 추정합니다.

## 2. 단답형 연습 문제

**문 1. AR(1) 모델 $Y(t) - \mu = \phi(Y(t-1) - \mu) + \epsilon(t)$에서 이 시계열이 약정상성(Weakly Stationary)을 갖기 위한 계수 $\phi$의 조건은 무엇인가?**
*   **정답:** $|\phi| < 1$

**문 2. 모델 선택 시 사용되는 BIC 지표에서 모델의 복잡성(파라미터 수)이 증가할수록 BIC 값은 어떻게 변화하는 경향이 있는가?**
*   **정답:** 페널티 항으로 인해 증가함 (최소화하는 것이 목표임).

**문 3. AUD.USD 1분봉 데이터를 AR(10) 모델로 분석하여 예측 가격이 현재 가격보다 높을 때 매수하고 낮을 때 매도하는 전략을 실행했을 때, 소스 텍스트에서 언급된 연간 수익률은 얼마인가?**
*   **정답:** 158% (단, 중간 가격으로 체결된다는 가정하에 저지연 실행 프로그램이 필요함).

**문 4. VAR(p) 모델이 단일 AR(p) 모델과 차별화되는 가장 큰 특징은 무엇인가?**
*   **정답:** 다변량 시계열을 다루며, 각 변수가 자신의 과거 값뿐만 아니라 다른 변수의 과거 값과도 연관됨.

**문 5. 상태 공간 모델에서 은닉 상태의 시간적 진화를 규정하는 방정식의 명칭은 무엇인가?**
*   **정답:** 상태 전이 방정식 (State Transition Equation).

## 3. 심화 학습을 위한 논술형 문제

**논술 1. 시계열 모델에서 복잡성 관리와 과적합(Overfitting)의 위험성에 대해 논하시오.**
*   **핵심 요약:** ARMA나 VAR 모델에서 시차($p, q$)가 커지거나 상태 공간 모델에서 추정해야 할 파라미터가 늘어나면 모델의 설명력은 높아질 수 있으나 과적합의 위험이 커집니다. 소스 텍스트에서는 이를 방지하기 위해 BIC와 같은 정보 기준을 사용하거나, 상관관계가 0이라는 제약을 두어 파라미터 수를 줄이는 등의 노력이 필요함을 강조합니다. 특히 칼만 필터의 예시에서 훈련 데이터에 대한 과도한 적합이 성과를 왜곡할 수 있음을 유의해야 합니다.

**논술 2. 주식 시장에서 섹터 중립(Sector-Neutral) 전략을 구축할 때 VAR 모델이 어떻게 활용될 수 있는지 설명하시오.**
*   **핵심 요약:** VAR 모델은 컴퓨터 하드웨어 업종과 같이 상관관계가 높은 주식 그룹의 수익률을 동시에 예측하는 데 적합합니다. 예측된 각 주식의 수익률($r_i$)과 업종 전체의 평균 예측 수익률($\langle r \rangle$)의 차이를 이용하여 비중을 배분함으로써, 시장 전체의 움직임에 대한 노출을 줄이고 종목 간 상대적 성과에 집중하는 전략을 수립할 수 있습니다.

**논술 3. 이동평균(Moving Average)을 은닉 변수로 간주하여 상태 공간 모델을 설계하는 논리를 서술하시오.**
*   **핵심 요약:** 표준적인 이동평균은 고정된 시차를 사용하여 계산되지만, 실제 시장의 '진정한' 평균 가격은 직접 관찰할 수 없는 은닉 상태로 볼 수 있습니다. 이를 위해 상태 전이 방정식에서는 평균값이 시간에 따라 진화한다고 가정하고, 측정 방정식에서는 실제 관찰되는 가격이 이 은닉된 이동평균에 노이즈가 섞인 형태라고 정의함으로써 더 유연한 필터링(예: 칼만 필터)이 가능해집니다.

## 4. 주요 용어 사전 (Glossary)

*   **약정상성 (Weakly Stationary):** 시계열의 평균과 분산이 시간에 따라 일정하며, 공분산이 시간 자체가 아닌 시차에만 의존하는 상태입니다.
*   **이노베이션 (Innovation):** 모델로 설명되지 않는 무작위 오차 항($\epsilon$)을 의미하며, 보통 평균이 0인 가우시안 노이즈로 가정됩니다.
*   **공적분 (Cointegration):** 각각은 비정상적(non-stationary)인 시계열들이 선형 결합을 통해 정상적(stationary)인 관계를 형성하는 현상입니다.
*   **칼만 필터 (Kalman Filter):** 노이즈가 포함된 관찰 값을 바탕으로 시간에 따라 변하는 시스템의 은닉 상태를 최적으로 추정하는 선형 재귀 필터입니다.
*   **오차 수정 행렬 (Error Correction Matrix - C):** VEC 모델에서 변수 간의 장기적인 균형 관계와 평균 회귀 속도를 나타내는 행렬입니다.
*   **입찰-매도 바운스 (Bid-Ask Bounce):** 실제 가치 변화 없이 매수와 매도 호가 사이에서 가격이 오가는 현상으로, 가짜 평균 회귀 신호를 생성할 수 있어 주의가 필요합니다.
*   **헤지 비율 (Hedge Ratio):** 두 자산(예: EWA와 EWC) 사이의 공적분 관계를 이용한 거래에서 포트폴리오의 위험을 중립화하기 위해 설정하는 비율입니다.
